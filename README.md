# TapBox
TapBox est une mini‑application web conçue pour être légère, rapide et amusante. Elle affiche des boutons interactifs qui réagissent instantanément lorsque l’utilisateur appuie dessus. L’app fonctionne sur tous les navigateurs modernes et peut être installée sur l’écran d’accueil de l’iPhone pour une expérience similaire à une vraie application iOS
